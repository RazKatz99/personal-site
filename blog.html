Building My First AI-Assisted Portfolio

February 23, 2026

For my DATA 305 assignment, I was tasked with building a personal website hosted on GitHub Pages, acting as both a resume and a blog. The twist? I had to use a Generative AI assistant to help me research and build it.

Lately, I've been spending a lot of time wrapping my head around C++ and Python syntax for different classes and projects. Building this site was a completely different experience. Instead of grinding through logic and debugging line by line, I acted more like a project manager. I decided on the architecture (Plain HTML/CSS) and directed the AI on the design, layout, and content structure.

It was fascinating to see how quickly a prompt could turn my actual CV into a fully styled webpage.
